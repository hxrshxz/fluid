Fluid Automation Queue

Safety policy: no PR creation/open actions in this UI. Keep explicit user confirmation before any PR operation.

How to Use

Follow this exact flow each time so queue updates stay predictable.

  1. Refresh queue to load latest issue parsing and state.
  2. Pick chunk from the table using Select.
  3. Run dry-run / execute using Daily Dry Run, then run real chunk work outside this UI.
  4. Update status for the selected chunk with status, PR URL, and notes.
